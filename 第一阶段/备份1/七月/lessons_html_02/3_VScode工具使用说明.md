## vscode 常用快捷键  
1. 文档目录切换  ctrl + b    
2. 插件扩展      ctrl + shift + x    
3. 关闭当前选项卡    ctrl + w    
4. 打开文件       ctrl + p  查找文件    
5. 分屏          ctrl + 1/2/3    
6. 文件切换       ctrl + tab 
7. 在当前行之后插入一行  ctrl + 回车
8. 在当前行之前插入一行  ctrl + shift + 回车
9. 多行同时输入          alt  + 光标点击
10. 移动代码       用alt + 上下方向键
11. 整行删除       shift + delete （mac  cmd + shift + k）
